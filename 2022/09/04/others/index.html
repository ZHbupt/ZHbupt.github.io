Node

nvm

作用：管理不同项目的node版本

  • node -v 查看node版本

  • nvm install lts/… 安装相对版本

  • nvm install v…. 安装绝对版本

  • nvm use …. 在当前控制台应用…版本

  • nvm alias a b 将a改为b版本

eg：nvm alias default lts/fermium （default为默认所有控制台使用版本）
